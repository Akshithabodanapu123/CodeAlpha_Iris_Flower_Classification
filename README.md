# Iris Flower Classification

## Project Overview
This project uses Machine Learning to classify Iris flowers into three species:
- Setosa
- Versicolor
- Virginica

## Dataset
The Iris dataset is loaded using Scikit-learn.

## Technologies Used
- Python
- Google Colab
- Scikit-learn
- Pandas
- NumPy

## Steps Performed
1. Loaded the Iris dataset.
2. Split data into training and testing sets.
3. Trained a Random Forest Classifier.
4. Evaluated model accuracy.
5. Predicted flower species.

## Result
The model successfully classified Iris flowers and achieved high accuracy.
